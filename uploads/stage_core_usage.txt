Stage core preparation usage

1) Build normalized index from synced Yandex Disk folder:
   python3 /workspace/scripts/firmware_catalog_ingest.py \
     --source "/home/ubuntu/Yandex.Disk/firmware_base" \
     --out-dir "/workspace/uploads/yadisk_ingest_run"

2) Build Stage1 core preparation artifacts:
   python3 /workspace/scripts/stage_core_builder.py \
     --index-csv "/workspace/uploads/yadisk_ingest_run/firmware_catalog_index.csv" \
     --out-dir "/workspace/uploads/stage_core_run"

3) Stage core output:
   - stage_core_families.csv
   - stage_core_backlog.csv
   - stage_core_operation_profiles.json
   - stage_core_summary.txt

Notes:
- This step prepares family readiness and operation templates only.
- Binary transformation logic must be implemented as a separate module.
